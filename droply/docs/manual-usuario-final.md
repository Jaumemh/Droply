# Manual de usuario final

## 1. Crear acceso
1. Abre Droply.
2. Introduce tu correo.
3. Recibe el codigo OTP.
4. Escribe el codigo de 6 digitos.

## 2. Subir un archivo
1. Entra al tablero.
2. Pulsa `Subir archivo`.
3. Selecciona un archivo.
4. Espera la barra de progreso hasta completar.

## 3. Compartir
1. Abre la tarjeta del archivo.
2. Pulsa `Compartir`.
3. Copia el enlace generado.
4. Envia el enlace al receptor.

## 4. Gestionar carpetas
1. Pulsa `Carpeta`.
2. Crea una carpeta nueva.
3. Entra en la carpeta deseada.
4. Arrastra mentalmente tu organizacion usando `Mover` en los archivos.

## 5. Mover archivos
1. Abre la card del archivo.
2. Pulsa `Mover`.
3. Elige carpeta destino.
4. Confirma el cambio.

## 6. Eliminar archivos
1. Pulsa `Eliminar`.
2. Droply borra el objeto de Storage y marca el archivo en la base de datos.

## 7. Abrir un enlace compartido
1. Abre el enlace recibido.
2. Verifica que el enlace no este caducado.
3. Pulsa `Descargar archivo`.
