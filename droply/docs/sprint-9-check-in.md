# Sprint 9 - Check-in semanal

## Progreso
- La jerarquia de carpetas sigue funcionando por `parent_id`.
- Ya se pueden mover archivos entre carpetas.
- El borrado de archivos sincroniza Storage y base de datos.
- La UI respeta `owner_id` y muestra solo el contexto actual.

## Experiencia de organizacion
- El usuario percibe Droply como un espacio ordenado, no como una lista plana.
- Las acciones de organizar estan cerca del contenido.
- La navegacion por carpetas es consistente y predecible.

## Demo funcional
- Crear una carpeta dentro de otra.
- Mover un archivo a otra carpeta.
- Eliminar un archivo y verificar que deja de aparecer.
- Navegar por niveles y comprobar que cada vista solo muestra su contenido.
